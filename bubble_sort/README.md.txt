Bubble sort algorithm is used to sort the lists.
The way this algorithm works is that it selects the two initial values of the list, then if the first value is greater than the second, it changes their place, and if it is not greater, it does not change.
It does this two by one for all members of the list and finally prints the sorted list.
It is interesting to know that this algorithm is also used to sort a list of letters.

الگوریتم مرتب سازی حبابی برای مرتب کردن لیست ها استفاده میشود.
شیوه کار این الگوریتم به این شکل است که دو مقدار اولیه لیست را انتخاب میکند سپس اگر مقدار اولی ازدومی بزرگتر بود،جای آنها را تغییر میدهد و اگر بزرگتر نبود تغییری اعمال نمیکند.
 این کار دو به دو برای تمامی اعضای لیست انجام میدهد و در آخر لیست مرتب شده را چاپ میکند .
جالب است بدانید که این الگوریتم برای مرتب کردن لیستی از حروف نیز مورد استفاده قرار میگیرد.
